This Firefox addon makes the tabbar vertical. Nothing else.
Toggle tabbar visibility with contextual menu or "ctrl + F8".

Styling is kept at a minimum to accomodate themes.
Styling can be improved further with userchrome.css.
A minimalist sample file is included at "extras/userChrome.css".

Packaged xpi releases here:
http://spenibus.net/files/gitbin/vertical-tabs-firefox-addon/
